How It Works

# 1. Register AI agent
agent = register_ai_agent(
  name="claude-opus",
  capabilities=["reasoning", "code_gen"]
)
# Returns: did:honestly:agent:claude-opus

# 2. Generate ZK proof of reputation
proof = get_agent_reputation(
  agent_id="claude-opus",
  threshold=40 # Prove > 40 without revealing score
)

# 3. Verify the proof
verify_zk_proof(proof)
# Returns: {"verified": true, "nullifier_valid": true}

Technical Innovation

First production implementation of zero-knowledge proofs for AI agent authentication

  • Novel ZK Circuit: Level3Inequality.circom for reputation threshold proofs
  • Identity Binding: Poseidon hashing prevents proof forgery
  • Standards Compliant: W3C DIDs + Verifiable Credentials
  • Production Cryptography: Groth16 (same as Ethereum ZK rollups)
  • Replay Protection: Nullifier tracking with Redis persistence

We extended standard ZK-SNARK circuits (age verification) to AI agent reputation - a novel contribution.
